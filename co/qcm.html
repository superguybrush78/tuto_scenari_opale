Question à Choix Multiple

Exemple

  • Clique droit dans la colonne Explorateur puis Créer un item...

  • Dans la liste Exercices auto-évalués, sélectionnez Question à Choix Multiple

Vous pouvez renommer le fichier .quiz qui sera créer ou garder le titre par défaut, puis Créer.

  • 1 Titre accroche⚓

    Exemple

    Entrez ici : L'origine du mot

  • 2 Énoncé⚓

    Exemple

    Entrez ici : De quelles racines d'origine grecque provient le mot photographie ?

  • 3 Choix⚓

    Exemple

    Entrez maintenant le libellé et l'explication associée des 3 réponses possibles :

    1. Libellé : photo

      Explication : Oui, le préfixe « photo- » (φωτoς, photos : lumière, clarté) — « qui procède de la lumière », « qui utilise la lumière » ;

    2. Libellé : gra

      Explication : Non...

    3. Pour créer une 3ème réponse, cliquez sur la réponse 2 puis cliquez sur le qui apparaît en bas à gauche de la question et Réponse.

      Libellé : graphie

      Explication :

      Oui, le suffixe « -graphie » (γραφειν, graphein : peindre, dessiner, écrire) — « qui écrit », « qui aboutit à une image ».

    4. Nous allons cocher les bonnes réponses :

      Réponse 1 : dans la liste de choix Solution, séléctionnez : Coché

      Réponse 2 : dans la liste de choix Solution, séléctionnez : Non Coché

      Réponse 3 : dans la liste de choix Solution, séléctionnez : Coché

  • 4 Explication globale⚓

    Comme pour le bloc énoncé, vous avez la possibilité ici d'ajouter plusieurs types de contenu à la suite. Nous allons ici ajouter une vidéo complémentaire.

    Exemple

    • Dans le champs par défaut, entrez : Le substantif féminin « photographie » (photography) a été proposé par John Herschel dès mars 1839.

    • Cliquez sur les trait horizontaux , puis sur le et enfin sélectionnez Ressource

    • Glissez la vidéo d'illustration John Herschel.mp4 sur l'icône d'insertion

    • Dans Mode de publication (web et diaporama), sélectionnez Embarquée (la vidéo sera intégré au document, Zoom signifie qu'un lien vers la vidéo sera affiché)

    • Dans Consigne (pour le web et diaporama), entrez : Ci-dessous une vidéo complémentaire sur John Herschel

    • Enregistrez votre travail

  • 5 Aperçu⚓

    Exemple

    Cliquez sur l'onglet Aperçu

    Cliquez sur Correction pour visualiser les champs entrés ainsi que la vidéo complémentaire.
